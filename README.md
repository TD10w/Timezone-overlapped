# Timezone Overlap

Timezone Overlap helps distributed teams see where their working hours meet without doing timezone math by hand.

## What it does

- Add cities or timezones to a shared overlap board
- Compare working hours on a visual timeline
- Highlight the best shared meeting window
- Drop pins on an interactive globe
- Translate one exact time across selected zones

## Run locally

```bash
npm install
npm run dev
```

Then open the local URL printed by Vite.

## Build

```bash
npm run build
npm run preview
```

## Stack

React, Vite, JavaScript, CSS, and browser timezone APIs.

## Status

An early product experiment exploring clearer timezone coordination for remote teams.
